# GRAPE Publication-Quality Tree Visualizations\n\nThis directory contains both ASCII and publication-quality visualizations of phylogenetic trees for major language families.\n\n## Visualization Types\n\nFor each language family, GRAPE provides:\n\n1. **ASCII Trees**: Text-based representations (`*_formatted.txt`)\n2. **Publication PNG**: High-resolution raster images (`publication/*.png`)\n3. **Publication SVG**: Scalable vector graphics (`publication/*.svg`)\n4. **Newick Format**: Standard phylogenetic format (`*.newick`)\n\n## Linguistic Best Practices\n\nPublication visualizations follow linguistic phylogenetic standards:\n\n- **Color coding** by established linguistic subgroups\n- **Clear typography** with readable language names\n- **Branch length display** showing evolutionary distances\n- **Comprehensive legends** for language group identification\n- **High resolution** (300 DPI) suitable for publication\n- **Multiple formats** (PNG, SVG) for different use cases\n\n## Language Families\n\n### Romance\n\n- **Languages**: 43\n- **Dataset**: `resources/language_families/romance.tsv`\n- **Tree height**: 3.000\n-
